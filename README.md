# wthr
wthr lets you view weather conditions through the [OpenWeatherMap API](https://openweathermap.org/api)

### Configuration
create your config in `<your_config_dir>/wthr/config.yml`

```yaml
api:
  key: "your OpenWeatherMap API key"

location:
  # lat / lon
  latitude:  00.000000
  longitude: 00.000000

preferences:
  # metric / imperial
  unit: "imperial"
```
